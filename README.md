# PIXEL FORGE

一个完全在浏览器本地运行的像素画优化工具。支持色彩量化、区域清理、限定色数、像素物理重构、背景抠除、透明描边、精灵图集分析、调色板与 PNG 导出。

## 本地运行

直接双击 `index.html` 即可使用。为了获得完整的文件夹保存能力，建议在本目录启动一个静态服务器：

```powershell
npm start
```

然后访问 <http://127.0.0.1:8765>。也可以直接双击 `index.html` 使用主要功能。

所有图片处理均在浏览器中完成，不会上传到服务器。现代 Chromium 浏览器还可通过左侧的文件夹按钮选择本地输出目录。
